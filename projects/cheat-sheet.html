CSS Positioning


Inline element Flow with content (<span>, <em>, <strong>)
Block elements Live on their own lines (<div>, <p>, <h1>)
Float Takes element out of normal flow and moves it as far left as possible.
position:static elements fall in normal flow.
position:fixed elements fixed relative to the browser window
position:absolute elements positioned relative to parent element
position:relative positioned relateive to itself

More info, see my post on positioning elements with CSS.
